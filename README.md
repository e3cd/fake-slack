## FakeSlack - Slack clone built using React hooks and Firebase realtime storage

### Technology stack
- React(with hooks)
- React Context API
- Firebase
- Material UI
- CircleCI

### Features
- Create and star channels 
- Instant messaging to users and channels
- Upload images
- User authentication

#### Run Locally
```js
> git clone https://github.com/e3cd/fake-slack.git
> yarn install
> yarn start
